    What is a Trie?

    A Trie (Prefix Tree) is a tree data structure used for efficient retrieval of a key in a dataset of strings. Unlike binary trees, nodes don't store their keys—their position in the tree defines the key.

    Complexity

    Insert / Search: O(L), where L is the length of the word. Extremely fast since it's independent of the total number of words (N).
    Memory Structure: Each node typically allocates an array of pointers (size 26 for English alphabet) and a boolean flag.
